3.4. Zoom

SRec allows to adjust the size of the tree and stack elements.

For that, the option Configuration / Zoom configuration must be selected. The zoom about the offered views can be graduated.

If zoom effect is not desired anymore, Si se desea retirar el efecto de zoom sobre la visualización, it is enough to press the 'Cancel' button and the zoom values will turn to 0.

When the size of the cells is the requeried, press the 'Accept' button for exit the dialog.

At the right side, there are three buttons which will let to extend and to reduce the graph in a 5%. The third button adjusts the graph in the panel.
