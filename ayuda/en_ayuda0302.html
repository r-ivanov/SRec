3. Visualizations configuration

3.2. Hidding of historical data and subcalls

The application gives the possibility of choosing between supporting previous steps information in the visualization or in being attenuating it or eliminating it. The format in that the attenuated values appear is configurable for the user.

On the other hand, the application allows to choose between showing the subcalls that are generated when a jump occurs on the present call, or not to show the generated subtree. This tree will be as long as the current configuration does not eliminate of the visualization the historical nodes.

In order to change these configurations it is necessary to select to the option Configuration / Animation options and to select in the second panel the wished action about the historical values. The changes will take place immediately in the visualization. Once "Aceptar" is pressed the dialogue picture will disappear maintaining the configuration chosen.
