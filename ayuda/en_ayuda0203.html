2. Visualizations

2.3. Load visualizations

SRec allows to load a visualization from disk. For loading a visualization, the option File / Load animation must be selected.

The user must select the XML file that contains the wanted animation. After that, the aplication reads all the configuration parameters, all the values of the running trace and shows them suitably represented.

Once the visualization is loaded, it can be used and configurated with freedom.
