2.2.2. Random generation of parameters values

To generate in a random way the values for the parameters, it is necessary to press the button "Generate" of the dialog for introducing of values.

User can select the parameters will take random values and limit the minimum and maximum possible values. Is not allowed any maximum value minor than the corresponding minimum value. The type of this values must be the same that the parameter type.

For arrays and matrices is allowed the introduction of dimensions size. If the user does not fill this value, it will take a random value.
