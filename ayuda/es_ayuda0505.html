5.5. Control de uso

SRec permite registrar el uso que se hace de la aplicación con fines meramente estadísticos de cara a mejorar la usabilidad de la aplicación en posteriores versiones.

Así, SRec genera un fichero textual, visible por un lector de texto plano, que almacena información de fechas, horas, e interacciones activadas durante el uso.

Este fichero no almacena ningún tipo de información sensible, ni tampoco la aplicación realiza ninguna conexión automática con ningún servidor para el envío de ésta u otra información.

Para activar o desactivar la creacióny actualización del archivo de registro de uso, basta seleccionar la opción Configuración / Habilitar archivo LOG.
