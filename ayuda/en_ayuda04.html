4. Reading and editing code

It's easy reading and editing the Java code of a algorithm with SRec. User only must to select the option File / Edit. For reading or editing the Java code of the process classs, ths option File / Edit class must be selected (the changes in the Java code will have effect when the class was reprocessed).

The buttons what appear in the editor window are:

Open: Opens other file which code will be howed in the editor window.

Save: Saves the text of the window in the file.

Cut: Cuttes the selected text and stores in the clipboard.

Copy: Copies the selected text and stores in the clipboard.

Paste: Pastes the text stored in the clipboard.
