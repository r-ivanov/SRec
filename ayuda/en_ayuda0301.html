3. Visualizations configuration

3.1. Showed data

The application allows to choose what information must appear in the visualization. This way, in a visualization can be showed the in parameters values and the return value, only the in parameters values or only the return value.

The option Configuration / Animation options must be selected for configurating this options. The changes will take effect instantaneously.

User can choose what methods and parameters stay visible in the animation by means of the option Configuration / Animation options.
