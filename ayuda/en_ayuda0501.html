5. More options

5.1. Java virtual machine selection

This option allows SRec to compile Java codes, so this option must be correctly configurated.

In order to configure adequately this option, Configuración / Java virtual machine must be selected. The user must write the local path where is located the Java virtual machine (it is necessary to have installed the JDK -Java kit of development-, is not enough with having the JRE -Java runtime environment-). In particular, is necessary to provide the complete path of the file "java.exe".

SRec allows to browse in the file system for selecting the file "java.exe".
