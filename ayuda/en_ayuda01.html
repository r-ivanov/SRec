1. Introduction to SRec

SRec allows to visualizate in a graphic, simple and intuitive form the running of recursive algorythms using activation trees. SRec offers others sights like the control stack, the execution trace and the algorythm code. SRec 1.2 is able to offer specific views for designed algorythms under the "Divide and Conquer" technique.

During visualizations, user can select what information SRec shows (parameters, methods...). The application let users interact with the views.

A lot of options about visualizations can be configurated and SRec allows save to disk and load visualizations (so you don't need to generate them once in each session) with all the configuration parameters.

The algorythm code can be opened for reading and editing, so it's very easy and quick to correct errors or to introduce modifications in the code.

SRec can show several visualizations simultaneously; it allows to compare runnings and to interpret results (in this version this feature isn't enabled).
