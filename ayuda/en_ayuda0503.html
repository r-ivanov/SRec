5. More options

5.3. Language

SRec allows to be used in multiple languages. By default, SRec's distribution offers Spanish and English. To configure the language of the window, menus and dialogs of SRec is very easy: the option Configuration / Language must be selected.

In the dialog, the user can see the current selected language, and a list of available languages. Once the user has selected the wished language, the 'Accept' button must be pressed.

After that, SRec shows a info message.
