2.2.1. Parameters values

Before the selected method running, the user must introduce the values for the in parameters. The correct form of each parameter type is:

Byte: a number in the range (-128..127).

Short: a number in the range (-32768..32767).

Int: a number in the range (-2147483648..2147483647).

Long: a number in the range (-9223372036854775808..9223372036854775807).

Float: a number with optional decimal part (between the integer part and the decimal part must be '.'. Scientific notation is not allowed.

Double: a number with optional decimal part (between the integer part and the decimal part must be '.'. Scientific notation is not allowed.

Char: an only one alphanumeric character.

String: set of alphanumeric characters.

Boolean: the words "true" or "false" must be typed for true and false values, sespectively.

For introducting arrays, it's needed to write '{' at the beginning and '}' at the end. Each element of the array must be separated of others elements by ','. White spaces are ignored.

Correct example for arrays:

Integer array (int): { 400, 74, 28, -823 }

Float point array (float): { 4.7, 7, 2.777778, -8.7 }

A matrix is considered a array of arrays. A set of arrays must be handled like elements of other array.

Correct example for matrices:

Integer matrix (int): { { 400, -434, 46 }, {4100, 1228, -991} }

Float point matrix (float): { {14.1, 177.11}, {2.777778, -8.7}, { 5.666, 6.555} }

To introduce multiple values for each parameter, values must be specified with commas, or through ranges for integer values X..Y

Examples for multiple values introduction:

Multiple String values: "one","two","three"

Multiple float values: 2.4,3.7,2.9

Range of integer values: 1..5
