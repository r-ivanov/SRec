SRec - Error de Ayuda

Se ha producido un error, no se encontró el recurso requerido. Pulse el botón 'i' para ir al índice de Ayuda. Si encuentra de nuevo este error al pulsar 'i', quiere decir que la ayuda en este idioma no está disponible.

____

SRec - Help Error

An error occured, SRec could not find the required resource. Press the 'i' button for going to the help index. If you find this error page again after pressing 'i', it means that help is not available for this language.

____
