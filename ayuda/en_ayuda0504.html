5.4. Configurations handling

SRec offers many options that allow to the user to manage different configurations, which can be loaded from disk when the user wishes. These options are:

Configuration / Restore configuración: Load the default configuration, which can be defined by the user. If a visualization is open in this moment, changes will take effect instantaneously about it.

Configuration / Open configuration: Load a previously saved configuration from disk. If a visualization is open in this moment, changes will take effect instantaneously about it.

Configuration / Save configuration: Save to disk the current configuration, it would be loaded in future sessions.

Configuration / Save default configuration: Save to disk the current configuration as default configuration. It would be loaded in future sessions like default configuration.
